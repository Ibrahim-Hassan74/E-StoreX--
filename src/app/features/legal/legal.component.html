1. Project Disclaimer

This website (E-StoreX) is a portfolio project created by Ibrahim Hassan for demonstration and educational purposes only. It is not a real commercial e-commerce store. No real products are sold, and no real payments are processed (Stripe is in Test Mode).

2. Data Collection

We collect minimal data to demonstrate functionality:

  • Account Info: Name, Email (for registration demo).
  • Cart & Wishlist: Stored locally or in database for user session demo.
  • Forms: Messages sent via the Contact form are processed by EmailJS.

3. Third-Party Services

This project integrates with the following services:

  • Stripe: Payment processing (Test Mode).
  • EmailJS: Email delivery service.
  • Google & GitHub: OAuth authentication.
